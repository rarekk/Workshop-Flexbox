### 🎯 1. จุดประสงค์การปฏิบัติงาน
1. เพื่อให้นักเรียนสามารถเขียนชุดคำสั่ง CSS Flexbox ในการจัดวางองค์ประกอบแบบ 1 มิติได้
2. เพื่อให้นักเรียนสามารถสร้างแถบนำทาง (Navbar) ได้อย่างถูกต้องตามมาตรฐาน Responsive
3. เพื่อให้นักเรียนตระหนักถึงการเขียนโค้ดอย่างเป็นระบบ โดยละเว้นการเดาสุ่มตัวเลข (Guess-and-check)

### 📚 2. คำสั่งพื้นฐานที่สำคัญ (CSS Flexbox)
ในการจัดรูปแบบหน้าเว็บไซต์ ให้นักเรียนศึกษาและเลือกประยุกต์ใช้คำสั่งพื้นฐานเหล่านี้:
```css
display: flex; /* เปิดใช้งาน Flexbox สำหรับกล่องแม่ (Container) */
justify-content: space-between; /* จัดกระจายให้ห่างกันซ้าย-ขวาสุด */
justify-content: center; /* จัดให้อยู่กึ่งกลางแนวนอน */
align-items: center; /* จัดให้อยู่กึ่งกลางแนวตั้ง */
gap: 20px; /* กำหนดระยะห่างระหว่างชิ้นส่วนลูก (โดยไม่ต้องใช้ margin) */
```

### 🛠️ 3. แบบฝึกหัด (ขั้นตอนการปฏิบัติงาน)
1. ให้สร้างไฟล์ `index.html` และสร้างโครงสร้าง HTML สำหรับแถบนำทาง (Navbar) **ด้วยตนเอง** ซึ่งประกอบด้วย **"โลโก้ (Logo)"** อยู่ฝั่งซ้าย และ **"เมนูนำทาง (Home, About, Contact)"** อยู่ฝั่งขวา
2. เขียนชุดคำสั่ง CSS เพื่อจัดรูปแบบ Navbar โดยมี **ข้อกำหนด** ดังต่อไปนี้:
   * **ต้องใช้** ชุดคำสั่ง `display: flex` และจัดการตำแหน่งด้วย `justify-content` หรือ `align-items`
   * **ห้ามใช้** คำสั่ง `float: left` หรือ `float: right` โดยเด็ดขาด
   * **ห้ามใช้** การกะระยะด้วยตัวเลขสุ่ม เช่น `margin-left: 300px;` หรือ `margin-right: 50%;`
3. ตกแต่งสีสันของพื้นหลัง ตัวอักษร และปุ่มเมนูให้สวยงามตามความคิดสร้างสรรค์
4. ตรวจสอบความถูกต้องโดยการย่อ-ขยายหน้าจอเบราว์เซอร์ องค์ประกอบต้องไม่ผิดเพี้ยน
