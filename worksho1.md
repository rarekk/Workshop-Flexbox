ใบงาน Workshop: จัด Layout เว็บไซต์ยุคใหม่ด้วย CSS Flexbox 📦
เป้าหมายของใบงานนี้:

รู้วิธีดาวน์โหลดโค้ดเริ่มต้น (Clone) จาก GitHub ของครู

เลิกใช้ float และเลิกสุ่มตัวเลข margin ในการจัดเรียงหน้าเว็บ

จัดของให้อยู่กึ่งกลางเป๊ะๆ ได้ในไม่กี่บรรทัด!

🚀 ขั้นตอนที่ 0: เตรียมไฟล์ทำงาน (Clone Git Repository)
ก่อนเริ่มเขียนโค้ด เราจะดึงไฟล์โครงสร้างพื้นฐานที่ครูเตรียมไว้ให้ลงมาที่เครื่องคอมพิวเตอร์ของเราก่อน

วิธีทำ:

เปิดโปรแกรม Terminal (หรือ Command Prompt / Git Bash)

พิมพ์คำสั่งเพื่อเข้าไปยังโฟลเดอร์ที่ต้องการเก็บงาน เช่น หน้า Desktop:

Bash
cd Desktop
พิมพ์คำสั่ง git clone ตามด้วย URL ของครู (ระวังเรื่องการเว้นวรรค):

Bash
git clone https://github.com/ชื่อผู้ใช้ของครู/ชื่อrepoของครู.git
(หมายเหตุ: นำลิงก์ที่ครูให้ในห้องเรียนมาใส่แทนที่ลิงก์ด้านบน)

เมื่อโหลดเสร็จแล้ว ให้เข้าไปในโฟลเดอร์งานที่เพิ่งโหลดมา:

Bash
cd ชื่อrepoของครู
เปิดโฟลเดอร์นี้ในโปรแกรม VS Code เพื่อเริ่มเขียนโค้ด:

Bash
code .
📦 การทดลองที่ 1: จัดกล่องเรียงแนวนอน (บอกลา Float และ Margin มั่วๆ)
ปัญหาที่เจอบ่อย: เวลาเราอยากให้กล่อง 3 ใบเรียงติดกัน เรามักจะติดนิสัยใช้ float: left; พอระยะมันชิดไปก็กะตัวเลขใส่ margin-right: 15px;... พอดูในมือถือ พัง! กล่องตกขอบ!

วิธีแก้ด้วย Flexbox: เราจะสั่งให้ "กล่องแม่" (Container) จัดการเรียงและเว้นระยะลูกๆ ให้เองแบบอัตโนมัติ

📝 ภารกิจ: ให้เปิดไฟล์ index.html แล้วพิมพ์โค้ดด้านล่างนี้ลงไป ทดลองลบและเพิ่มเครื่องหมาย /* ... */ ในส่วนของ CSS เพื่อดูความเปลี่ยนแปลง



```html
<!DOCTYPE html>
<html lang="th">
<head>
    <meta charset="UTF-8">
    <title>Flexbox แบบเรียงแนวนอน</title>
    <style>
        /* ตกแต่งกล่องให้ดูง่าย */
        .box {
            width: 100px;
            height: 100px;
            background-color: #ff4757;
            color: white;
            font-size: 2rem;
            text-align: center;
            line-height: 100px;
            border-radius: 10px;
        }
        
        /* ---------------------------------------------------
           โซนเขียนโค้ด Flexbox (กล่องแม่)
           --------------------------------------------------- */
        .box-container {
            background-color: #f1f2f6;
            padding: 20px;
            border: 2px dashed #ccc;
            
            /* 1. เสกให้กล่องแม่เป็น Flexbox เพื่อเรียงของแนวนอน */
            display: flex; 
            
            /* 2. สั่งเว้นระยะห่างทุกกล่อง 20px เท่ากันเป๊ะ (ไม่ต้องสุ่ม margin แล้ว!) */
            gap: 20px; 
        }
    </style>
</head>
<body>

    <h2>การทดลองที่ 1: เลิกสุ่ม Margin</h2>
    <div class="box-container">
        <div class="box">1</div>
        <div class="box">2</div>
        <div class="box">3</div>
    </div>

</body>
</html>
```

👀 สังเกตผลลัพธ์: เมื่อกดเซฟและดูหน้าเว็บ กล่องทั้ง 3 ใบจะเรียงต่อกันในแนวนอน และมีช่องว่าง (gap) ตรงกลางเท่ากันเป๊ะๆ โดยที่เราไม่ต้องบวกเลขกะระยะเองเลย!

🎯 การทดลองที่ 2: จัดของให้อยู่ "กึ่งกลาง" หน้าจอแบบ 100% เป๊ะ!
ปัญหาที่เจอบ่อย: อยากให้ข้อความหรือกล่องอยู่ตรงกลางหน้าจอเป๊ะๆ บางคนใช้ margin-top: 150px; margin-left: 200px; สุ่มตัวเลขไปเรื่อยๆ พอเปลี่ยนขนาดจอคอม อ้าว... เบี้ยวอีกแล้ว!

วิธีแก้ด้วย Flexbox: ใช้คำสั่งคู่หูมหัศจรรย์ คือ justify-content (จัดกลางแนวนอน) และ align-items (จัดกลางแนวตั้ง)

📝 ภารกิจ:
สร้างไฟล์ใหม่ชื่อ center.html แล้วพิมพ์โค้ดด้านล่างนี้ลงไป เพื่อดูความมหัศจรรย์ของการจัดกึ่งกลาง

```html
<!DOCTYPE html>
<html lang="th">
<head>
    <meta charset="UTF-8">
    <title>Flexbox จัดกึ่งกลาง</title>
    <style>
        /* ---------------------------------------------------
           โซนเขียนโค้ด Flexbox (พื้นที่จัดกึ่งกลาง)
           --------------------------------------------------- */
        .center-zone {
            /* กำหนดความสูงของพื้นที่ให้เต็มจอ (Viewport Height) */
            height: 100vh; 
            background-color: #2ed573;
            color: white;

            /* เริ่มใช้งาน Flexbox */
            display: flex;
            
            /* จัดให้อยู่กึ่งกลางแนวนอน (ซ้าย-ขวา) */
            justify-content: center; 
            
            /* จัดให้อยู่กึ่งกลางแนวตั้ง (บน-ล่าง) */
            align-items: center; 
        }

        h1 {
            background-color: rgba(0,0,0,0.5);
            padding: 20px 40px;
            border-radius: 15px;
        }
    </style>
</head>
<body style="margin: 0;"> 

    <div class="center-zone">
        <h1>✨ สวัสดี Flexbox! ✨</h1>
    </div>

</body>
</html>
```
