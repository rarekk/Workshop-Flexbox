🛋️ Workshop 1: จัดห้องนั่งเล่นด้วย Flexbox
ยินดีต้อนรับสู่ภารกิจแรก! ใน Workshop นี้ นักเรียนจะได้รับบทเป็น Interior Designer ที่ต้องใช้พลังของ CSS Flexbox ในการจัดวางเฟอร์นิเจอร์ภายในห้องนั่งเล่นให้เข้าที่เข้าทางตามโจทย์ที่ได้รับ

🎯 เป้าหมายการเรียนรู้
ฝึกใช้ display: flex เพื่อเปิดใช้งาน Flexbox Container

ฝึกการจัดแนวนอนด้วย justify-content

ฝึกการจัดแนวตั้งด้วย align-items

ฝึกการกำหนดระยะห่างด้วย gap

📂 โครงสร้างโปรเจกต์ (Starter Code)
ภายใน Repository นี้จะประกอบด้วยไฟล์หลัก 2 ไฟล์:

index.html: โครงสร้างห้องนั่งเล่น (.living-room) และเฟอร์นิเจอร์ (.item) 3 ชิ้น (ตู้เย็น, โซฟา, ทีวี)

style.css: ไฟล์สำหรับเขียน CSS (นักเรียนจะแก้ไขไฟล์นี้เป็นหลัก)

🛠️ ภารกิจ (Missions)
นักเรียนต้องแก้ไขไฟล์ style.css ในส่วนของ .living-room เพื่อผ่านด่านทั้ง 3 ระดับ ดังนี้:

🚩 ภารกิจที่ 1: เปิดประตูเข้าห้อง (Basic)
โจทย์: จงทำให้เฟอร์นิเจอร์ทั้ง 3 ชิ้น เรียงต่อกันเป็นหน้ากระดาน (แนวนอน) และอยู่ กึ่งกลางห้อง พอดี

คำใบ้: สั่งงานที่ .living-room โดยใช้ display: flex; และเลือกค่า justify-content ที่ดึงของมาไว้ตรงกลาง

🚩 ภารกิจที่ 2: เว้นระยะห่างเพื่อทางเดิน (Spacing)
โจทย์: จงจัดให้เฟอร์นิเจอร์ กระจายตัวออกจากกัน โดยให้ชิ้นแรกชิดซ้ายสุด และชิ้นสุดท้ายชิดขวาสุด เพื่อให้เหลือพื้นที่เดินตรงกลางกว้างๆ

คำใบ้: ลองเปลี่ยนค่าใน justify-content เป็นแบบที่ผลักวัตถุไปไว้ที่ขอบ (Space)

🚩 ภารกิจที่ 3: จัดระเบียบแนวตั้ง (Alignment)
โจทย์: ทำให้เฟอร์นิเจอร์ทั้งหมดลงมาอยู่ กึ่งกลางห้องในแนวตั้ง และให้แต่ละชิ้นมี ระยะห่างกัน 20px

คำใบ้: เพิ่มคำสั่ง align-items สำหรับแนวตั้ง และใช้ gap สำหรับระยะห่าง
